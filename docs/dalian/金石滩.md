# 金石滩
大连金石滩国家旅游度假区的黄金海岸绵延4.5公里，宽100～200米，是北方最大的海岸，可同时容纳10万人。海滩之上建了高杆广场灯，形成了夜间灯光浴场（至22:00）。沿浴还建成了一条高水准的滨海观光路——西起金石国际会议中心。冬至金石高尔夫俱乐部，全长12.8公里，同大连市内滨海路不同，这条路濒临大海，是一条游人可以近海亲海的观光路。
黄金海岸的游乐项目除传统的海水浴以外，还囊括潜水、帆船、沙滩摩托、沙滩排球等活动项目，成为海上运动的大本营。而海滩篝火烧烤、“乘游艇、观岸壁风光、看野生海豚”和“巨型仿真鲸鱼”等特色项目，更为海滩平添几分神秘和浪漫。
黄金海岸有国内第一个全部使用防紫外线设施的健康型海滨浴场，上千个沙滩凉亭、帐篷、阳伞都是用无污染的草或者 是用防太阳黑子材料制作，构成一道道沙滩奇观。海岸沙质金黄、颗粒均匀，水质属一类海域，优于大连市中心地区海水浴场。1999年8月，中国国家主席江泽民曾下海畅游并题词“大连金石滩”

金石滩国家旅游度假区位于辽东半岛南端的黄海之滨，毗邻大连经济技术开发区、大连保税区和大连双D港，距离大连中心50公里、大连国际机场45公里、沈大高速公路30公里由香炉礁至金石滩的轻轨电车，成为大连至金石滩的旅游新干线，交通便捷。金石滩陆地面积62平方公里、海域面积58平方公里，海岸线长30公里。三面环海，四季分明海域不淤不冻，属于暖温带半湿润气候，素有“东北小江南”之称，是中国北方理想海滨旅游度假胜地。

1988年国务院批准金石滩为国家重点风景名胜区，1992年国务院批准在金石滩建设国家旅游度假区。区内旅游资源极其丰富：这里有6～9亿年的金石园、玫瑰园、恐龙园、南秀园、鳌滩等五大景区和100多处景点。其中，龟裂石被地质专家称天下第一石；大鹏展翅、神龟寻子、刺猬觅食、恐龙探海等奇石景观被世界地质学家誉为“凝固的动物世界”、“海上石林”、“神力雕塑公园”和“天然地质陈列馆”等。金石滩的自然条件非常适合开发观光度假旅游产业，如天然植被和大面积风景林是北方狩猎理想场所；200多米宽的海水浴场绵延4.5公里，沙软滩平，水质清洁，是辽东半岛最佳浴场之一；周边有丰富的鱼类资源，是理想的垂钓场所，2002年4月，金石滩浴场被国家海洋局评定为全国15个健康型海水浴场之一。并先后被评为“大连市十大景区”，“辽宁省50佳景”，“辽宁省文明景区及全国首批4A级旅游景区”

金石滩旅游度假区基础设施建设基本完备，全区绿化副嘎46.7%，旅游接待设施粗具规模，建成金石高尔夫球场、金石国际会议中心、影视艺术中心、金石蜡像馆、中华武术馆、婚礼殿堂、金石园、狩猎俱乐部、外事公寓、影视艺术学校、枫叶国际学校、国际游艇俱乐部、鲜花大世界、模特剧场、金石广场、天鹅湖等项目。金石滩的温泉资源具有非常高的养生保健价值，建设中的温泉度假村项目以温泉洗浴、理疗为特色的度假休闲中心。金石滩海滨观光路贯穿东西两个半岛和中部海滨4.5公里浴场，长达12.8公里，在修葺一新的海滨浴场海滩上增建了高杆广场灯。建设了温泉度假村、金石马术场、金石谷乡村俱乐部、光彩神州农业示范苑，并积极推进室内游乐场、五星级酒店、渔人码头、海上夜总会等招商。东北最大的主题公园项目也于2003年5月开工建设，预示着金石滩未来发展的希望和辉煌。
## 金石蜡像馆
金石蜡像馆位于金石滩中心区域大道北侧。该蜡像馆汇集了众多世界名人，展出蜡像100余尊，规模在国内最大。这里的蜡像是世界著名华人蜡像师张守祥雕塑的，他根据重大历史事件和真实情节，共划分为“历史长河”、“风云人物”、“中美建交”、“香港回归”、“奥运之光”等13个主题。特别是“香港回归”记录的是1982年邓小平与英国首相撒切尔夫人会晤的历史时刻。邓小平以他非凡的智慧，在解决香港问题上提出了“一国两制”的伟大构想，为澳门和台湾问题的最终解决树立了不朽典范。1997年香港回归的历史时刻也是蜡像馆正式开馆的日子。它向世人奉献了这组具有历史意义的蜡像作品，以示大连人民的祝贺旨意。
